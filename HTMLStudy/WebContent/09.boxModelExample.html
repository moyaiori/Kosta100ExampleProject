박스(컨테이너) 모델과 보더 설정

한번에 지정 : border속성


세분화시켜서 지정 :
border-width, border-color, border-style속성


위,아래만 선 표시 :
border-top, border-bottom 속성


위,아래 점선 표시 :
border, border-bottom-width, border-top-width 속성


선두께 다양하게 지정 :
아래쪽은 thin, 위쪽은 thick, 왼쪽은 medium, 오른쪽은 1px;


점선의 모양 비교 :
dotted는 동그란 모양, dashed는 사각형 모양


선의 색상을 다르게 표현 :
아래쪽은 #CC6600, 위쪽은 #FF0066


padding은 안쪽여백, margin은 바깥여백
안쪽여백 0px, 바깥여백 0px


배경이미지 : background-image속성


위치지정 : position





>>border-style의 종류 비교<<

none solid double groove ridge


inset outset dashed dotted hidden
